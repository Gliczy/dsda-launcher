#### These are the parameters that work on dsda-doom

| Parameter                                        | Description                                                |
| :-------------                                   | :----------                                                |
| -iwad <doom.wad>                                 | Specifies the IWAD to load                                 |
| -file <av.wad>                                   | Specifies a PWAD to load                                   |
| -file <av.wad>                                   | Specifies a PWAD to load                                   |
